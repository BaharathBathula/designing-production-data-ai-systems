# Chapter 11: Real Time Fraud Detection Pipeline

## Overview

Fraud detection is one of the most demanding production data and AI use cases. Systems must process transactions instantly while balancing risk, customer experience, and regulatory obligations.

## Core Topics

- Streaming transaction ingestion
- Real time feature engineering
- Low latency model inference
- Dynamic risk scoring
- False positive reduction
- Monitoring and drift detection
- Global scale resilience

## Key Principles

- Speed with accuracy
- Low latency decisions
- Continuous learning
- Reliable operations under pressure

## Key Takeaway

Fraud systems succeed when data engineering, machine learning, and operations work as one platform.
